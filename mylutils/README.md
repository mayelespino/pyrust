# mylutils

This is a collection of file utilities. It is currently under active development and not quite ready for primetime


## How to install and run

### Pip install

```
pip install mylutils
```

### How to use

```
import mylutils

for line in mylutils.read_txt("test.txt"):
    print(line)

```
